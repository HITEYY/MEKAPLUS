# MEKAPLUS MOD (Minecraft 1.21.1 / Forge)

MEKAPLUS mod implementation for progression in a modpack with **AE2 + Mekanism + Create**.

## Implemented Core Features

1. **Starlight**
   - New epic material item.
   - Crafting recipe integrates AE2/Mekanism/Create advanced resources through tags.

2. **Soul Head + Boss**
   - `Soul Head` craft: Wither Skeleton Skull + Starlight.
   - Right-click with Soul Head on soul-sand T structure to summon **Soul Revenant** boss.
   - Boss stats target: 500 HP / 15 attack damage.

3. **Antigear**
   - Boss guaranteed drop (1x).

4. **Antimatter Synthesizer**
   - New high-tier block + block entity scaffold.
   - Current logic: consumes 8x Antigear to produce 1x Vibranium every 200 ticks.

5. **Vibranium**
   - High-tier material item generated by Antimatter Synthesizer logic.
   - Used in:
     - `128x Speed Module`
     - `Damage Boost Module`

## Notes

- Placeholder textures/models are included.
- AE2/Mekanism/Create integration is recipe/tag based (optional dependencies).
- Machine UI/energy network integration is scaffold-level and can be expanded.

## Project Structure

- `src/main/java/com/hiteyy/mekaplus/` - main mod code
- `src/main/resources/assets/mekaplus/` - models/lang/textures
- `src/main/resources/data/mekaplus/` - recipes/loot
