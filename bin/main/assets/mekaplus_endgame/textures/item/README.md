# Item Textures

Place 16x16 pixel PNG textures here with the following names:

- `temporal_alloy_ingot.png` - Epic rarity ingot with temporal/time-based theme
- `infinite_energy_matrix.png` - Epic rarity matrix/grid pattern with energy theme
- `radiance_shadow_amalgam.png` - Rare rarity combining light and dark elements
- `antimatter_stabilization_matrix.png` - Rare rarity with antimatter/quantum theme
- `dimensional_fusion_core.png` - Epic rarity core/orb with dimensional rifts
- `temporal_catalyst.png` - Epic rarity catalyst with time distortion effects

## Suggested Color Schemes:
- **Temporal Alloy Ingot**: Purple/Blue with clock-like patterns
- **Infinite Energy Matrix**: Bright cyan/white with grid overlay
- **Radiance-Shadow Amalgam**: Half gold/yellow, half dark purple/black
- **Antimatter Stabilization Matrix**: Dark with red/pink accents
- **Dimensional Fusion Core**: Multi-colored with portal-like swirls
- **Temporal Catalyst**: Glowing purple with time-warp effects
